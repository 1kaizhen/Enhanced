Foundation

Motion

Transitions are short (150ms) for hover states. The two exceptions: dock tiles lift on a 250ms springy curve, and the settings cog rotates 45° in 200ms. Nothing else animates — the UI relies on stillness for its calm feel.

Transitions in use — hover any element
Hover
.15s linear · background / color / border-color
Dock lift
.25s cubic-bezier(.34,1.56,.64,1) · translateY + scale
Cog spin
.2s · transform: rotate(45deg)
Principles
  • · Default to .15s for any color / background / border feedback.
  • · Use the springy curve only on transforms — and only on the dock.
  • · Don't animate state changes in lists (todos, bookmarks, notes). Full re-render is fine.
  • · No loaders. If something is slow, fix the slowness, don't dress it.
