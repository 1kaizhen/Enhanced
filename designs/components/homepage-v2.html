Tokens · Radii & corner shape

Every rounded surface pairs border-radius with corner-shape: squircle. In supporting browsers (Chrome with the flag) the corners are smoothed continuously; everywhere else the property is ignored and you get circular fallback radii.

Component · info-chip

A compact 12px-text pill that shows ambient information (weather, workspace). Always carries a leading icon or 6px colored dot. Used in the top-left of the homepage.

Component · action-pill (with count)

Primary action target in the top-right cluster. The trailing dark count badge is optional — leave it out when the count is zero or not relevant. Uses 13px label vs the info-chip's 12px so it reads as more interactive.

Component · clock row + meta line

The meta line above the clock is always a horizontal sequence of all-caps strings joined by middle-dots. PM/AM sits as a smaller secondary cap aligned to the top of the digits.

Component · search bar (engine badge variant)

The leading orange square is a colored engine badge — the user's current search engine, swappable via the caret. Replaces the previous magnifier-only engine selector. Right side keeps the white-on-glass go button.

Component · bookmarks grid

Inline 6-column grid that replaces the bottom dock. Each tile is a 56×56 squircle with a 2-letter initial cap in 800-weight and a 11px label below in soft white. Mix tile colors freely — there is no required mapping.

Bookmarks 6
Component · bottom-left cluster

Small settings cog (10px radius squircle, 32×32) followed by keyboard-shortcut hints. Each shortcut is a pair of <kbd> tags with a label.

⌘N Notes · ⌘T Tasks
Component · dropdown panel · Notes

Anchors below the Notes pill via position: absolute on a .action-with-dropdown wrapper. 300px wide, max-height 70vh, scrolls internally. Note mini-cards reuse the sticky-note palette (yellow / pink / blue / green / purple / peach) with dark text for legibility.

Component · side pane · To-do

Full-height pane that slides in from the right when the To-do pill is clicked. The rest of the page is dimmed and blurred. Pane width is 420px, padding 28px. Contains: header (title + subtitle + close), green progress bar with %, quick-add input, filter row with count badges + Clear-done link, and the task list.

Priority colors in this pane use a brighter scheme than the rest of the system — red #ef4444 (high), amber #f4a23c (medium), blue #3b82f6 (low). The done indicator is green #22c55e, matching the progress bar.

Layout · page grid

Body is a CSS Grid with rows auto 1fr auto — the top bar pins to the top, the center stage fills the middle and centers content with flex, the bottom-left cluster sits at the bottom. Page padding is 22px / 28px (vertical / horizontal). No floating absolute-position chrome.
